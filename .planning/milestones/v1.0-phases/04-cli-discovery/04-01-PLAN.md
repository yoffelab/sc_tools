    **Create `sc_tools/cli/discovery.py`** with these components:

    1. **Module-level constants:**
       ```python
       _CLICK_TYPE_MAP = {
           "text": "string",
           "integer": "integer",
           "float": "number",
           "boolean": "boolean",
           "path": "string",
           "uuid": "string",
           "choice": "string",
       }
       # Commands to exclude from discovery output (meta/discovery commands, not pipeline operations)
       _EXCLUDED_COMMANDS = {"version", "list-commands", "describe", "schema"}
       ```

    2. **`_walk_commands(target_app: typer.Typer) -> dict[str, click.Command]`:**
       - Call `typer.main.get_command(target_app)` to get Click Group
       - Create `click.Context(click_app)`
       - Iterate `click_app.list_commands(ctx)`
       - For each name: if `isinstance(cmd, click.Group)`, walk subcommands with `cmd.list_commands(grp_ctx)` and key as `f"{name} {sub_name}"`. Use `cmd.get_command(grp_ctx, sub_name)` to resolve.
       - If not a Group (leaf command at top level), key as bare `name`
       - Skip stub groups with no subcommands (integrate, celltype)
       - Filter out names in `_EXCLUDED_COMMANDS`
       - Return `dict[str, click.Command]`

    3. **`_param_to_schema(param: click.Parameter) -> dict`:**
       - Map `param.type.name` through `_CLICK_TYPE_MAP` (default to "string")
       - Add `"description": param.help` if param.help is not None
       - Add `"default": param.default` if param.default is not None
       - Add `"enum": list(param.type.choices)` if `hasattr(param.type, "choices") and param.type.choices`
       - Return the schema dict

    4. **`_command_to_entry(name: str, cmd: click.Command) -> dict`:**
       - Build params JSON Schema object: `{"type": "object", "properties": {...}, "required": [...]}`
       - For each param in `cmd.params`: skip if `param.name in ("help",)` or `(param.is_eager and not param.expose_value)`.
       - If param is a `click.Argument` and `param.required`, add to required list
       - If param is a `click.Option` and `param.required`, add to required list
       - `"group"`: first token of name if space in name, else None
       - `"description"`: `cmd.help or cmd.short_help or ""`
       - Return `{"name": name, "group": group, "description": description, "params": params_schema}`

    5. **`list_commands` command** (registered on app as top-level):
       - Decorated with `@cli_handler`
       - Accept `json_output: bool = typer.Option(False, "--json", help="JSON output (always JSON, flag for consistency)")`
       - Import `app` lazily inside function body: `from sc_tools.cli import app as main_app`
       - Call `_walk_commands(main_app)`
       - Build entries list via `_command_to_entry` for each command
       - Sort by name
       - Return `CLIResult(status=Status.success, command="list-commands", data={"commands": entries, "count": len(entries)}, provenance=Provenance(command="list-commands"), message=f"{len(entries)} commands available")`

    6. **`describe` command** (registered on app as top-level):
       - Decorated with `@cli_handler`
       - Accept `command_name: str = typer.Argument(help="Command name (space-separated, e.g. 'qc run')")`
       - Import `app` lazily, call `_walk_commands`
       - If `command_name` not in commands dict: raise `SCToolsUserError` with message `f"Unknown command: {command_name}"`, suggestion listing available command names
       - Build entry via `_command_to_entry`
       - Add `"output_schema": CLIResult.model_json_schema()` to the entry dict
       - Return `CLIResult(status=Status.success, command="describe", data=entry, provenance=Provenance(command="describe"), message=f"Schema for '{command_name}'")`

    7. **`schema` command** (registered on app as top-level):
       - Decorated with `@cli_handler`
       - Import `app` lazily, call `_walk_commands`
       - Build commands dict: `{name: _command_to_entry(name, cmd) for name, cmd in sorted(commands.items())}`
       - Remove "name" key from each entry (redundant — name is the dict key)
       - Get CLIResult schema: `cli_schema = CLIResult.model_json_schema()`
       - Extract `$defs` from cli_schema, add `"CLIResult"` key pointing to the CLIResult properties
       - Build data: `{"schema_version": "1.0", "sc_tools_version": _get_version(), "commands": commands_dict, "output": {"$ref": "#/$defs/CLIResult"}, "$defs": defs}`
       - Return `CLIResult(status=Status.success, command="schema", data=data, provenance=Provenance(command="schema"), message=f"Full CLI schema: {len(commands_dict)} commands, {len(defs)} shared types")`
