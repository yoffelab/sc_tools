<objective>
Implement pseudobulk differential expression via PyDESeq2 with CLI command (SCI-01).

Purpose: Pseudobulk DE is the statistically correct approach for multi-sample single-cell experiments. This replaces ad-hoc cell-level Wilcoxon tests that inflate type-I errors by treating cells as independent observations.

Output: `sc_tools/tl/de.py` module with aggregation + DE pipeline, `sc_tools/cli/de.py` with `sct de run` command, per-celltype CSV results.
</objective>

       `aggregate_pseudobulk(adata, subject_key, celltype_key, min_cells=10, layer=None) -> dict[str, tuple[pd.DataFrame, pd.DataFrame]]`:
       - Extract raw counts: try `adata.layers[layer]` if layer specified, then `adata.layers.get('counts')`, then `adata.raw.X`, then `adata.X`. Validate values are non-negative integers (round and cast if close to int).
       - For each celltype: subset adata, group by subject_key, sum counts per subject.
       - Filter subjects with < min_cells cells. Skip celltype if < 2 valid subjects.
       - Return {celltype: (count_df [subjects x genes, int], metadata_df [subjects x covariates])}.
       - Use scipy.sparse.issparse check, convert per-subject slices with .toarray() (per Pitfall 4: aggregate per-celltype subsets).

       `run_pseudobulk_de(adata, condition_key, *, subject_key="subject_id", celltype_key="celltype", formula=None, reference=None, min_subjects_per_group=3, min_cells_per_combo=10, layer=None, alpha=0.05) -> dict[str, pd.DataFrame]`:
       - Call aggregate_pseudobulk to get per-celltype counts.
       - For each celltype:
         - Check condition groups have >= min_subjects_per_group subjects each. Skip with warning if not.
         - Build design formula: if formula is None, auto-infer "~ {condition_key}". If batch-like columns (library_id, batch) exist in metadata, apply collinearity guard before auto-adding:
           - For each candidate batch column, check if it is collinear with the subject index: `metadata_df.groupby(subject_key)[batch_col].nunique().max() == 1` AND `metadata_df.groupby(batch_col)[subject_key].nunique().max() == 1` (1:1 mapping). If collinear, skip adding that column and log a warning: "Skipping '{batch_col}' from auto-formula: collinear with '{subject_key}' (1:1 design, would produce rank-deficient matrix)."
           - Only add non-collinear batch columns: "~ {condition_key} + batch" (per D-01).
         - Determine reference level: use `reference` param if given, else alphabetically first level.
         - Build contrast: [condition_key, tested_level, reference_level] where tested_level is the other level.
         - Run PyDESeq2: DeseqDataSet(counts, metadata, design), dds.deseq2(), DeseqStats(dds, contrast, alpha), stat_res.summary().
         - Rename columns to match D-03: gene (from index), log2FC (from log2FoldChange), pvalue, padj, baseMean.
         - Store in results dict.
       - Return {celltype: results_df}.

    3. Add `__all__` with `aggregate_pseudobulk` and `run_pseudobulk_de`.

       ```python
       de_app = typer.Typer(help="Differential expression commands")

       @de_app.callback(invoke_without_command=True)
       def de_callback(ctx: typer.Context) -> None:
           if ctx.invoked_subcommand is None:
               typer.echo(ctx.get_help())

       @de_app.command("run")
       @cli_handler
       def de_run(
           file: str = typer.Argument(..., help="Path to cell-typed h5ad"),
           condition: str = typer.Option(..., "--condition", "-c", help="Condition column in obs"),
           project_dir: str = typer.Option(".", "--project-dir", help="Project directory for output"),
           subject_col: str = typer.Option("subject_id", "--subject-col", help="Subject ID column"),
           celltype_col: str = typer.Option("celltype", "--celltype-col", help="Cell type column"),
           reference: str = typer.Option(None, "--reference", help="Reference level for contrast"),
           formula: str = typer.Option(None, "--formula", help="Custom design formula (e.g. '~ condition + sex')"),
           min_subjects: int = typer.Option(3, "--min-subjects", help="Min subjects per condition group"),
           min_cells: int = typer.Option(10, "--min-cells", help="Min cells per subject+celltype"),
           layer: str = typer.Option(None, "--layer", help="Count layer name override"),
       ) -> None:
       ```
