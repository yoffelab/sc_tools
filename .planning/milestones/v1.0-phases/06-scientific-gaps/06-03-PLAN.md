    1. Create `sc_tools/tests/test_marker_validation.py`:
       - Build a fixture: adata with 3 celltypes, set specific genes to have known expression per type.

       - Set one celltype to have very low expression of its markers (all below 0.1 threshold).
       - Test all behaviors listed above.

    2. Create `sc_tools/qc/marker_validation.py` implementing (per D-05, D-06, D-07):

       `compute_marker_validation(adata, celltype_key, marker_genes, *, threshold=0.1, top_n=5) -> tuple[pd.DataFrame, dict]`:
       - For each celltype in marker_genes:
         - Subset adata to cells of that type
         - For each marker gene: compute mean expression across cells of that type
         - If gene not in adata.var_names, set mean_expr=NaN
       - Build DataFrame: celltype, marker_gene, mean_expr, flagged
       - A celltype is flagged if ALL its canonical markers have mean_expr < threshold (per D-05)
       - Build summary dict: n_types_tested (celltypes in marker_genes that exist in data), n_flagged, total_cells
       - Return (validation_df, summary_dict)

       `render_marker_dotplot(adata, celltype_key, marker_genes, *, top_n=5) -> str`:
       - Use `scanpy.pl.dotplot` to render top_n markers per celltype
       - Return base64 PNG string via `fig_to_base64`
       - If scanpy not available, return empty string

    3. Export from `sc_tools/qc/__init__.py`: add `compute_marker_validation` to imports and `__all__`.
  </action>
