  <action>
    **Create `sc_tools/io/metadata.py`:**
    - Function `read_h5ad_metadata(path: str | Path) -> dict` that opens file via h5py (lazy imported), reads:
      - n_obs: via fallback chain (attrs._index_length -> obs/_index length -> first obs key length). Same pattern as inspect_checkpoint lines 510-523.
      - n_vars: same fallback chain on f["var"]
      - x_dtype: str(f["X"].dtype) if Dataset, str(f["X"]["data"].dtype) if Group (sparse)
      - x_sparse: isinstance(f["X"], h5py.Group)
      - obs_columns: [k for k in f["obs"].keys() if not k.startswith("__")]
      - obsm_keys: list(f["obsm"].keys()) if "obsm" in f else []
      - layer_keys: list(f["layers"].keys()) if "layers" in f else []
      - uns_keys: list(f["uns"].keys()) if "uns" in f else []
      - If sparse, also read nnz from f["X"]["data"].shape[0]
    - Return dict with all above fields.

    **Create `sc_tools/io/estimate.py`:**
    - Function `estimate_from_h5(path: str | Path) -> dict` that calls read_h5ad_metadata, then:
      - For dense X: x_bytes = n_obs * n_vars * np.dtype(x_dtype).itemsize
      - For sparse X: x_bytes = nnz * dtype_size + nnz * 4 + (n_obs + 1) * 4 (data + indices + indptr)
      - obs_bytes = n_obs * 200 (conservative per-cell estimate per RESEARCH.md)
      - var_bytes = n_vars * 100
      - base_mb = (x_bytes + obs_bytes + var_bytes) / (1024**2)
      - estimated_peak_mb = base_mb * 2.0 (overhead multiplier for processing intermediates)
      - Return dict: {n_obs, n_vars, x_dtype, x_sparse, x_mb, obs_mb, var_mb, base_mb, estimated_peak_mb}
    - Method-specific multiplier dict for estimate command (optional, used by Plan 02):
      METHOD_MULTIPLIERS = {"preprocess": 2.5, "de": 2.0, "qc": 1.5, "default": 2.0}

    **Create `sc_tools/io/gateway.py`:**
    - Enum `DataTier(str, Enum)` with values: T1_METADATA = "metadata", T2_SUMMARY = "summary", T3_FULL = "full" (per D-03)
    - Class `IOGateway`:
      - `read(self, path: str | Path, tier: DataTier, *, force: bool = False) -> dict | AnnData`:
        - T1: calls read_h5ad_metadata(path), returns dict
        - T2: calls anndata.read_h5ad(str(path), backed="r"), returns backed AnnData
        - T3: calls self._check_memory_guard(path) unless force=True, then calls anndata.read_h5ad(str(path)), returns full AnnData
      - `_check_memory_guard(self, path: str | Path) -> None`:
        - Calls estimate_from_h5(path) to get estimated_peak_mb
        - Calls psutil.virtual_memory().available / (1024**2) for available_mb
        - If estimated_peak_mb > available_mb * 0.8: raise SCToolsRuntimeError with message "Estimated memory ({estimated_peak_mb:.0f}MB) exceeds 80% of available RAM ({available_mb:.0f}MB). Use --force to override or reduce dataset size."
    - All heavy imports (h5py, anndata, psutil) are lazy (inside methods, not at module level) per CLI-08.

    **Create `sc_tools/io/__init__.py`:**
    - Exports: IOGateway, DataTier, read_h5ad_metadata (from metadata), estimate_from_h5 (from estimate)
    - Lazy imports (import inside functions or re-export via __all__) to avoid loading h5py at import time.

    - Use tmp_path fixture for cleanup.
