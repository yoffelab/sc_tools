**Create `sc_tools/errors.py`** with this exact hierarchy (per D-12, D-13):

```python
class SCToolsError(Exception):
    """Base exception for sc_tools."""
    category: str = "fatal"
    suggestion: str = ""
    exit_code: int = 3

    def __init__(self, message: str, *, suggestion: str = "", details: str | None = None):
        super().__init__(message)
        self.suggestion = suggestion or self.__class__.suggestion
        self.details = details

class SCToolsUserError(SCToolsError):
    """User-correctable error (bad args, missing file). Exit code 1."""
    category = "fixable"
    exit_code = 1

class SCToolsDataError(SCToolsError):
    """Data validation error (file exists but wrong format). Exit code 2."""
    category = "fixable"
    exit_code = 2

class SCToolsRuntimeError(SCToolsError):
    """Runtime error (OOM, computation failure). Exit code 3."""
    category = "retryable"
    exit_code = 3

class SCToolsFatalError(SCToolsError):
    """Unrecoverable error. Exit code 3."""
    category = "fatal"
    exit_code = 3
```

Each exception class has `category`, `suggestion`, `exit_code` as class attributes. The `__init__` accepts `message`, `suggestion` (keyword-only), and `details` (keyword-only, optional).

```python
from __future__ import annotations

from datetime import datetime, timezone
from enum import Enum
from typing import Any

from pydantic import BaseModel, Field

# Status enum per D-07
class Status(str, Enum):
    success = "success"
    error = "error"
    partial = "partial"
    skipped = "skipped"

# Error info per D-13, D-14
class ErrorInfo(BaseModel):
    category: str  # "retryable" | "fixable" | "fatal"
    suggestion: str
    details: str | None = None

# Minimal provenance per D-10
class Provenance(BaseModel):
    command: str
    timestamp: str = Field(default_factory=lambda: datetime.now(timezone.utc).isoformat())  # noqa: UP017
    sc_tools_version: str = Field(default_factory=_get_version)

# CLIResult envelope per D-06, D-08, D-09
class CLIResult(BaseModel):
    status: Status
    command: str
    data: dict[str, Any] = Field(default_factory=dict)
    artifacts: list[str] = Field(default_factory=list)
    provenance: Provenance
    message: str = ""
    error: ErrorInfo | None = None
    # Per D-08: structured failures for agent retry of partial results
    partial_failures: list[dict[str, Any]] | None = None
```

The `_get_version()` helper function must be defined BEFORE the `Provenance` class. It uses `importlib.metadata.version("sci-sc-tools")` with a try/except returning `"unknown"` if not installed. Use `# noqa: UP017` on the `timezone.utc` usage per project convention.

Use `model_config = ConfigDict(use_enum_values=True)` on CLIResult so JSON serialization uses string values not enum names.

Exit code mapping per D-15, D-16: 0=success (includes partial), 1=user error, 2=data error, 3=runtime error. This mapping is documented in a module-level docstring but the actual exit code logic lives in the CLI handler (Plan 02).
