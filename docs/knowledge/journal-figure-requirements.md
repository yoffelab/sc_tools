## 1. Nature (and Nature family)

**Source:** [Nature Research Figure Guide](https://research-figure-guide.nature.com/figures/preparing-figures-our-specifications/), [Nature Final Submission](https://www.nature.com/nature/for-authors/final-submission)

### Dimensions
- Single column: **89 mm** (3.5 in)
- Double column: **183 mm** (7.2 in)
- Maximum height: **170 mm** (for main figures, to allow legend space)
- Extended data max page: **183 x 247 mm**

### Resolution
- Photographic/raster: **300 dpi minimum**, 450 dpi recommended
- Line art: vector preferred (no minimum dpi if vector)

### File Formats
- **Preferred:** PDF, EPS (vector)
- Also accepted: AI, PSD, TIFF (300-600 dpi)
- File size max: 10 MB per figure

### Fonts
- **Sans-serif required:** Helvetica or Arial preferred
- Body text: **5-7 pt** at final size
- Panel labels: **8 pt bold, lowercase** (a, b, c), upright (not italic)
- Courier for amino-acid sequences

### Color
- Submit in **RGB** (auto-converted to CMYK for print)
- Must comply with WCAG 2.1 Level AA accessibility
- **Avoid:** red/green combinations, rainbow/jet colormaps
- Recommended: Wong/Okabe-Ito palette, viridis, cividis

### Line Width
- Minimum: **0.5 pt** (some sources say 1 pt at final size)

### Panel Labeling
- **Lowercase bold**: a, b, c (8 pt, upright, not italic)

### Figure Count
- Nature: typically 6-8 main figures, up to 10 Extended Data figures
- Extended Data: up to 10 figures + 10 tables
- Supplementary Information: no strict limit

### Prohibitions
- No background gridlines
- No drop shadows
- No fill patterns (use solid colors)
- No colored text in legends (use colored boxes with black text)
- No outlined/rasterized text (must remain editable)
- No rainbow colormaps
- Avoid 3D effects on charts

## 2. Cell (and Cell Press family)

**Source:** [Cell Press Figure Guidelines](https://www.cell.com/figureguidelines)

### Dimensions (2-column format: Cell, Cell Reports, Cancer Cell, etc.)
- 1 column: **85 mm** (3.35 in)
- 1.5 columns: **114 mm** (4.49 in)
- Full width: **174 mm** (6.85 in)

### Dimensions (1-column format: Chem, Joule, Matter, Med)
- 1 column: **112 mm** (4.41 in)
- Full width: **172 mm** (6.77 in)

### Resolution
- Color/grayscale: **300 dpi minimum**
- Black and white: **500 dpi minimum**
- Line art: **1,000 dpi minimum**

### File Formats
- **Preferred:** EPS (vector, all fonts embedded)
- Also accepted: JPEG, TIFF, PDF, AI
- Graphical abstracts: 1200 x 1200 px at 300 dpi

### Fonts
- **Avenir** preferred; Arial or similar sans-serif if unavailable
- Text: **6-8 pt** at print size
- Graphical abstract text: 8-12 pt

### Color
- Submit in **RGB**
- Colorblind-friendly palettes encouraged

### Panel Labeling
- **Uppercase**: A, B, C

### Figure Count
- Typically 7 main figures for Cell
- Supplementary figures: labeled as "Figure S1" etc.
- STAR Methods section for detailed methods

### Prohibitions
- No image manipulation that misrepresents data
- Gel/blot images must show full, uncropped versions in supplementary

## 3. Science (and Science family)

**Source:** [Science Instructions for Authors](https://www.science.org/content/page/instructions-preparing-revised-manuscript)

### Dimensions
- 1 column: **57 mm** (2.24 in)
- 2 columns: **121 mm** (4.76 in)
- 3 columns (full width): **184 mm** (7.24 in)

### Resolution
- All images: **300 dpi minimum** at final print size
- Line art: higher preferred; vector recommended
- Initial submission: 300 dpi acceptable

### File Formats
- **Preferred (in order):** PDF, EPS, AI (vector)
- Raster: TIFF (300 dpi minimum)
- Avoid: JPEG for line art

### Fonts
- **Helvetica** preferred (sans-serif)
- Consistent font throughout all figures

### Color
- Submit in **RGB**
- Free color in print and online

### Panel Labeling
- **Uppercase bold**: A, B, C
- Capitalize first letter of labels only (not every word)
- Units in parentheses, SI notation

### Figure Count
- Research articles: typically 4 main figures
- Reports: typically 4 figures
- Supplementary Materials: no strict figure limit

### Line Width
- Minimum: **0.5 pt** at final size

### Prohibitions
- No unnecessary decorative elements
- Avoid pattern fills when solid colors suffice

## 4. The Lancet

**Source:** [Lancet Artwork Guidelines](https://www.thelancet.com/pb/assets/raw/Lancet/authors/artwork-guidelines.pdf), [Lancet Info for Authors (2025)](https://www.thelancet.com/pb-assets/Lancet/authors/tl-info-for-authors-1690986041530.pdf)

### Dimensions
- Single column: **75 mm** (2.95 in)
- Double column: **154 mm** (6.06 in)
- Submit figures **20% larger** than final print size

### Resolution
- **300 dpi** at 120% of final size

### File Formats
- Accepted: PowerPoint, Word, SVG, TIFF, JPEG, EPS, PDF
- Vector preferred (Illustrator files welcome)
- **Note:** All figures are redrawn by in-house illustrators

### Fonts
- Consistent font size throughout
- Sans-serif recommended (Arial, Helvetica)

### Color
- Submit in **RGB**

### Panel Labeling
- All panels on a single page
- Consistent labeling style

### Figure Count
- Research articles: typically up to 5 figures
- All figure panels must fit on a single page

### Prohibitions
- No box outlines on graphs
- No titles within graphs/artwork
- Minimal figure legends

## 5. PNAS

**Source:** [PNAS Digital Art Guidelines](https://www.pnas.org/pb-assets/authors/digitalart-1675347574760.pdf)

### Dimensions
- 1 column: **87 mm** (3.42 in)
- Wide figures: **105-180 mm** (4.13-7.09 in)

### Resolution
- Halftones: **300 dpi minimum**
- Combinations (line + photo): **600-900 dpi**
- Line art: **1,000-1,200 dpi**

### File Formats
- **Preferred:** TIFF (LZW compression, not JPEG compression), EPS, PDF
- Vector text preferred

### Fonts
- **Standard fonts:** Arial, Helvetica, Times, Symbol, Mathematical Pi, European Pi
- Size: **6-8 pt minimum** at publication size
- Text labels at least **2 mm tall** when printed

### Color
- Submit in **RGB** (original color space)
- Do not convert to CMYK before submission

### Panel Labeling
- **Uppercase bold**: A, B, C
- Upper-left corner of each panel
- Consistent style across all figures

### Figure Count
- No strict limit on main figures (typically 6)
- SI appendix: max 10 files total (combined into single PDF, excluding videos)

### Line Width
- Minimum: recommended **0.5 pt** at final size

### Prohibitions
- No JPEG compression in TIFF files
- Avoid outlined/rasterized text

## 6. NEJM (New England Journal of Medicine)

**Source:** [NEJM Technical Guidelines for Figures](https://www.nejm.org/pb-assets/pdfs/Technical-Guidelines-for-Figures-1511366945697.pdf), [NEJM Author Center](https://www.nejm.org/author-center/new-manuscripts)

### Dimensions
- Not specified for authors -- **all figures are professionally redrawn** by NEJM design team after acceptance

### Resolution
- Black-and-white line art: **1,200 dpi**
- Initial submission: lower resolution acceptable for peer review

### File Formats
- Data visualizations: **editable vector files** (AI, EPS, SVG) strongly preferred
- BMP acceptable for black-and-white line art only
- Can embed in manuscript or upload separately

### Fonts
- Not strictly specified (figures are redrawn)
- Authors provide sketches, PowerPoint slides, or rough figures

### Color
- Full color available

### Panel Labeling
- Applied by NEJM design team during redrawing

### Figure Count
- Research articles: typically 4-5 figures
- Figures redrawn to journal style after acceptance

### Special Notes
- Authors collaborate with editorial production team on final figures
- No image manipulation: no features enhanced, obscured, moved, removed, or introduced
- Brightness/color/contrast adjustments must apply to entire image

## 7. eLife

**Source:** [eLife Author Instructions](https://elife-rp.msubmit.net/html/elife-rp_author_instructions.html)

### Dimensions
- Minimum physical width: **100 mm** (10 cm)
- Full-page figures: minimum **200 mm** (20 cm)
- No strict maximum specified

### Resolution
- **300 dpi minimum** for all figures

### File Formats
- Accepted: **TIFF** (8-bit), **JPG**, **EPS**, **AI**, **PDF** (vector only)
- Multi-page figures not supported
- Each component uploaded separately

### Fonts
- No specific font mandated
- Standard scientific convention (sans-serif recommended)

### Color
- Submit in **RGB**
- Follow Colour Universal Design (CUD) principles for accessibility

### Panel Labeling
- No strict case mandate; consistent style required
- Figure supplements: "Figure 1--Figure Supplement 1" format

### Figure Count
- No strict limit on main figures
- Unlimited figure supplements
- Source data files encouraged (under 100 MB each)

### Prohibitions
- No image adjustments that could lead to misinterpretation
- Brightness/contrast adjustments must be uniform across entire image and controls
- Combined images must show dividing lines
- Disable hidden layers before upload
- Minimize whitespace (especially in PDFs)

## Universal Best Practices

These apply regardless of target journal:

1. **Always use vector** for line art, graphs, and diagrams (PDF/EPS/AI)
2. **Always submit RGB** -- let the journal handle CMYK conversion
3. **Sans-serif fonts** (Helvetica or Arial) at 6-8 pt minimum
4. **300 dpi minimum** for any raster content at final print size
5. **Colorblind-safe palettes**: use viridis, cividis, or Okabe-Ito/Wong palette
6. **Avoid**: rainbow/jet colormaps, red-green only distinctions, 3D bar charts, background gridlines, drop shadows, pattern fills
7. **Scale bars** on microscopy images (not magnification factors)
8. **Line width minimum**: 0.5 pt at final print size
9. **Consistent style** across all figures in the manuscript
10. **No image manipulation** that alters scientific content
