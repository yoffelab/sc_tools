# IMC projects

Folder layout for Imaging Mass Cytometry (IMC) projects. Each subfolder is a project (e.g. aapc, bcg_bladder, brca-unet, colon_liver_meta, ggo-imc, ggo_human, lymph_dlbcl, pancreas_liver_meta).

Project-specific data, figures, results, and metadata are not committed (see repo root `.gitignore`). Only scripts, config (YAML), and small structural files under 1MB are tracked.
