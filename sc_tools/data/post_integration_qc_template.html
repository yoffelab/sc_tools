Metric definitions
Batch removal metrics (higher = better batch mixing):
• ASW Batch — Average silhouette width for batch labels (1 − |score|; penalizes batch-specific clusters)
• PCR — Principal component regression; variance in PCs explained by batch (1 − R²; lower batch influence = better)
• Graph Conn — Graph connectivity; fraction of cells in the largest connected subgraph per cell type (batch-mixed neighborhoods)
Bio conservation metrics (higher = better biological separation):
• ASW Cell — Average silhouette width for cell type labels (distinct cell types form tight clusters)
• ARI — Adjusted Rand Index; agreement between Leiden clusters and known cell types
• NMI — Normalized Mutual Information; information shared between clusters and cell types
Composite scores: Batch = mean(ASW Batch, PCR, Graph Conn). Bio = mean(ASW Cell, ARI, NMI). Overall = 0.6 × Batch + 0.4 × Bio.
